Tentang AnimeKu

Website informasi anime berbasis API.

Apa itu AnimeKu?

AnimeKu merupakan website pencarian informasi anime yang memanfaatkan Jikan API. Website ini memungkinkan pengguna mencari anime, melihat detail anime, melihat anime populer, serta mencari anime berdasarkan genre. Seluruh data yang ditampilkan berasal dari API, sehingga informasi selalu mengikuti data terbaru.

Fitur Website

Pencarian Anime

Cari anime berdasarkan judul.

Top Anime

Menampilkan anime dengan rating tertinggi.

Genre Anime

Anime berdasarkan kategori genre.

Detail Anime

Menampilkan informasi lengkap anime.

Teknologi

PHP HTML5 CSS3 JavaScript Fetch API Jikan API Font Awesome
